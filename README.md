Zala Temető Nyilvántartás Rendszer

Projekt Áttekintés
  - Ez a projekt egy temető nyilvántartási rendszer, amely célja egy falusi temető adatainak megtekintése és adminisztrációjának megkönnyítése.
  - A rendszer frontend és backend komponensekből all és lehetővé teszi a temető információinak hatékony kezelését.

Főbb Jellemzők
  - Felhasználóbarát felület: A rendszer könnyen használható és navigálható, lehetővé téve a felhasználók számára, hogy gyorsan hozzáférjenek a szükséges információkhoz.
  - Adatkezelés: A temetők adatainak kezelésére szolgáló funkciók.
  - Keresés és szűrés: Felhasználók gyorsan kereshetnek és szűrhetnek információkat a temetőben található nyughelyekről.
  - Adminisztrációs felület: Az adminisztrátorok számára külön admin felület áll rendelkezésre az adatok kezelésére.

Technológiai Stack
  - Frontend: React.jsx, CSS, HTML ( Modern és reszponzív felhasználói felület)
  - Backend: Laravel, MariaDB

Projekt Struktúra
  - /frontend: A frontend kód forrása.
  - /backend: A backend kód forrása.

Telepítés és Használat
